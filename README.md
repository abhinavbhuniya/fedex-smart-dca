# FedEx Smart DCA Control Tower

AI-powered platform to reimagine Debt Collection Agency (DCA) management for the FedEx SMART Hackathon.

## Problem
Manual handling of DCAs using Excel and email causes delays, weak governance, and poor recovery visibility.

## Solution
A centralized, AI-driven platform that:
- Predicts recovery probability
- Prioritizes overdue cases
- Enforces SLA-driven workflows
- Provides real-time dashboards

## Tech Stack
- Python
- Streamlit
- Scikit-learn

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
